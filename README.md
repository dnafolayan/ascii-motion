# ASCII Motion

A Python project for creating ASCII art animations.

## Prerequisites

-   Python 3.x
-   uv package manager

## Installation

1. Clone this repository:

```bash
git clone https://github.com/dnafolayan/ascii-motion.git
cd ascii-motion
```

2. Install dependencies using uv:

```bash
uv install
```

## Usage

Run the main script:

```bash
uv run main.py
```
